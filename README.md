# Library-management-system-project
The Library Management System is a Java-based application developed to automate and simplify the daily operations of a library. This project provides a user-friendly and efficient platform where users can easily manage their library activities through a secure and organized system.The application allows users to create an account and log in using their credentials. Once logged in, users can view the list of available books, search books by name, and check out books according to availability. Every book transaction is recorded along with the transaction time, allowing users and administrators to track borrowing history in an accurate and transparent way.The system is designed to reduce manual paperwork and improve efficiency in managing library records. It helps maintain a proper record of issued and returned books and ensures that all transactions are stored securely. The simple and clean interface makes it easy for users to navigate and perform actions without technical difficulty.

This project is suitable for schools, colleges, and small to medium-sized libraries that want a reliable digital solution for managing their library operations. It demonstrates the practical use of object-oriented programming concepts, database connectivity, and user authentication in Java.

Overall, the Library Management System provides a complete solution for handling library activities in a fast, secure, and organized manner while improving the overall user experience and record management process.

# Library Management System

## Project Overview
Ye project Java Swing par bana hua Library Management System hai.
Is mein login system, books management aur search system maujood hai.

## Features
- Login system
- Add & view books
- Search books
- Issue & return system

## Technologies Used
- Java
- Swing GUI
- MySQL
## How to Run the Project

Follow these steps carefully to run the project successfully:

### Step 1: Start XAMPP Server
1. Open XAMPP Control Panel
2. Start Apache and MySQL services

### Step 2: Import Database
1. Open browser and go to: http://localhost/phpmyadmin
2. Click on "Import"
3. Select the database file provided separately with the project
4. Click on "Go" to import the database

### Step 3: Open Project in IDE
1. Open NetBeans or IntelliJ IDEA
2. Open the project folder

### Step 4: Run Login System
1. First run the LoginForm class
2. Enter login credentials from the database
3. After successful login, the main system will open

### Important Note
Make sure XAMPP is running before starting the project.
Database must be imported first, otherwise login system will not work.
